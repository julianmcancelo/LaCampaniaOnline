# LaCampaniaOnline

Implementacion online de La Campania con Next.js 14, Socket.io y un motor de reglas propio en TypeScript.

## Stack

- Next.js 14
- Socket.io
- Tailwind CSS
- Framer Motion
- Zustand
- TypeScript

## Scripts

- `npm run dev`
- `npm run build`
- `npm run test`

## Estado

El proyecto incluye:

- lobby y salas multijugador
- motor de batalla alineado al reglamento oficial
- flujo de turnos por fases
- reconexion por `playerId`
- UI de tablero con drag and drop
